What is this?
It's a database, a JSON storage, available from the internets.
How does it work?
You use your api key to access the data. You use a different one for writing it. More on this here.
Is the data safe?
No! While you can protect it yourself, security is not a concern of this service. If you need a really secure DB, this is not the answer.
So anyone can write into my database?
Again, no. Only applications that know of the secret key will be able to write the data.
How much can I store?
5 documents, none larger than 1MB. For the time being.
Why would I even need this?
Are you in a hurry and don't want to set up a DB server? Can't? Want your app to be purely HTML, Js and CSS? Well, this might be the answer :)
